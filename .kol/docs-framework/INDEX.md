# kol-docs framework

The doc spec this repo's `docs/` conform to — three nested tiers (fm ⊂ md ⊂ lib), the canon packages copied from the dotfiles scaffold.

- [[kol-docs-fm/INDEX|kol-docs-fm]] — frontmatter contract + tag taxonomy.
- [[kol-docs-md/INDEX|kol-docs-md]] — one whole doc (archetypes, anatomy, examples).
- [[kol-docs-lib/INDEX|kol-docs-lib]] — the whole repo docs library (structure + Obsidian).

Re-run the copy from `~/.dotfiles/claude/packages/kol-docs/{kol-docs-fm,kol-docs-md,kol-docs-lib}` by hand if the canon packages change and this repo needs the update — there is no automated sync.
